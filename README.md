# crud-clubes-grp

Crud clubes its a Gran DT demo. in this demo we can see, add and delete teams of a teams list.


## Install and Run

```bash
  git clone https://github.com/GGrop/crud-clubes-grp
  npm install
  nodemon index.js
```

## Tech Stack

```bash
  Node.js
  Multer
  Handlebars
```
